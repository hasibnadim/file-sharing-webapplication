# File Sharing webApplication

## Integrated by Node.js, ejs, Express.js, MongoDB

## List Of api:

###### Home

> HOSTNAME/

- Method `GET`

###### Upload

> HOSTNAME/upload

- Method `POST`
- Form `filetitle:text` , `file:file`

###### get file

> HOSTNAME/:fileid

- Method `GET`

###### Download file

> HOSTNAME/download/:fileid

- Method `GET`
